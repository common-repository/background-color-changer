== Description ==
This is a simple plugin to change the background color, text color, and heading color of the theme. This plugin provides a customizer option in the theme.

Plugin Documentation: [https://wordpress.org/plugins/background-color-changer/](https://wordpress.org/plugins/background-color-changer/)

*   [About Author](https://github.com/raj009)


== You can change ==
*   Background Color (Unlimited colors).
*   Text Color (Unlimited colors).
*   Heading Color (Unlimited colors).


You can make my day by submitting a positive review on <a href="https://wordpress.org/plugins/background-color-changer/" target="_blank"><strong>WordPress.org!</strong></a></p>

= Usage =

* Go to your Customizer Option after installation and activation of the plugin. The Background Color Changer Plugin will be available there under the name Background Customization.


= Features =

  * Very easy installation
  * Flexible and easy to use for admin
  * Unlimited colors for the background, text, and heading


== Installation ==
1. Upload `background-color-changer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
